This is the API

integrations/getById

Get integration metadata by ID. Note: If you're interested in joining the beta, please reach out to your Pinterest account manager.

URL

https://api.pinterest.com/v5/integrations/{id}

HTTP Method

GET

Response Type

IntegrationRecord

Parameters

  • Path: id String

    Integration ID.

integrations/getList

Get integration metadata list. Note: If you're interested in joining the beta, please reach out to your Pinterest account manager.

URL

https://api.pinterest.com/v5/integrations

HTTP Method

GET

Response Type

integrations_get_list_200_response

Parameters

  • Query: bookmark String

    Cursor used to fetch the next page of items

  • Query: page_size Integer

    Maximum number of items to include in a single page of the response. See documentation on <a href='/docs/getting-started/pagination/'>Pagination</a> for more information.

integrationsCommerce/del

URL

https://api.pinterest.com/v5/integrations/commerce/{external_business_id}

HTTP Method

DELETE

Response Type

Parameters

  • Path: external_business_id String

    External business ID for the integration.

POST

Response Type

Parameters

integrationsLogs/post

This endpoint receives batched logs from integration applications on partner platforms. Note: If you're interested in joining the beta, please reach out to your Pinterest account manager.

URL

https://api.pinterest.com/v5/integrations/logs

HTTP Method

POST

Response Type

IntegrationLogsSuccessResponse

Parameters

  • Body: IntegrationLogsRequest IntegrationLogsRequest(IntegrationLogsRequest)

    Ingest log information from external integration application.
