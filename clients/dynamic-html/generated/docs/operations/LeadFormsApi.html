This is the API

leadForm/get

<strong>This feature is currently in beta and not available to all apps, if you're interested in joining the beta, please reach out to your Pinterest account manager.</strong> Gets a lead form given it's ID. It must also be associated with the provided ad account ID. For more, see <a class=\"reference external\" href=\"https://help.pinterest.com/en/business/article/lead-ads\">Lead ads</a>.

URL

https://api.pinterest.com/v5/ad_accounts/{ad_account_id}/lead_forms/{lead_form_id}

HTTP Method

GET

Response Type

LeadFormResponse

Parameters

  • Path: ad_account_id String

    Unique identifier of an ad account.

  • Path: lead_form_id String

    Unique identifier of a lead form.

leadFormTest/create

Create lead form test data based on the list of answers provided as part of the body. - List of answers should follow the questions creation order. <strong>This endpoint is currently in beta and not available to all apps. <a href='/docs/getting-started/beta-and-advanced-access/'>Learn more</a>.</strong>

URL

https://api.pinterest.com/v5/ad_accounts/{ad_account_id}/lead_forms/{lead_form_id}/test

HTTP Method

POST

Response Type

LeadFormTestResponse

Parameters

  • Path: ad_account_id String

    Unique identifier of an ad account.

  • Path: lead_form_id String

    Unique identifier of a lead form.

  • Body: LeadFormTestRequest LeadFormTestRequest(LeadFormTestRequest)

    Subscription to create.

leadForms/create

<strong>This feature is currently in beta and not available to all apps, if you're interested in joining the beta, please reach out to your Pinterest account manager.</strong> Create lead forms. Lead forms are used in lead ads and allow you to control what text appears on the lead form’ s description, questions and confirmation sections. For more, see <a class=\"reference external\" href=\"https://help.pinterest.com/en/business/article/lead-ads\">Lead ads</a>.

URL

https://api.pinterest.com/v5/ad_accounts/{ad_account_id}/lead_forms

HTTP Method

POST

Response Type

LeadFormArrayResponse

Parameters

  • Path: ad_account_id String

    Unique identifier of an ad account.

  • Body: LeadFormCreateRequest List(LeadFormCreateRequest)

    List of lead forms to create, size limit [1, 30].

leadForms/list

<strong>This feature is currently in beta and not available to all apps, if you're interested in joining the beta, please reach out to your Pinterest account manager.</strong> List lead forms associated with an ad account ID. For more, see <a class=\"reference external\" href=\"https://help.pinterest.com/en/business/article/lead-ads\">Lead ads</a>.

URL

https://api.pinterest.com/v5/ad_accounts/{ad_account_id}/lead_forms

HTTP Method

GET

Response Type

lead_forms_list_200_response

Parameters

  • Path: ad_account_id String

    Unique identifier of an ad account.

  • Query: page_size Integer

    Maximum number of items to include in a single page of the response. See documentation on <a href='/docs/reference/pagination/'>Pagination</a> for more information.

  • Query: order String

    The order in which to sort the items returned: “ASCENDING” or “DESCENDING” by ID. Note that higher-value IDs are associated with more-recently added items.

  • Query: bookmark String

    Cursor used to fetch the next page of items

leadForms/update

<strong>This feature is currently in beta and not available to all apps, if you're interested in joining the beta, please reach out to your Pinterest account manager.</strong> Update lead forms. Lead ads help you reach people who are actively looking for, and interested in, your goods and services. The lead form can be associated with an ad to allow people to fill out the form. For more, see <a class=\"reference external\" href=\"https://help.pinterest.com/en/business/article/lead-ads\">Lead ads</a>.

URL

https://api.pinterest.com/v5/ad_accounts/{ad_account_id}/lead_forms

HTTP Method

PATCH

Response Type

LeadFormArrayResponse

Parameters

  • Path: ad_account_id String

    Unique identifier of an ad account.

  • Body: LeadFormUpdateRequest List(LeadFormUpdateRequest)

    List of lead forms to update, size limit [1, 30].
