This is the API

audiences/create

Create an audience you can use in targeting for specific ad groups. Targeting combines customer information with the ways users interact with Pinterest to help you reach specific groups of users; you can include or exclude specific audience_ids when you create an ad group. <p/> For more, see <a class=\"reference external\" href=\"https://help.pinterest.com/en/business/article/audience-targeting\" target=\"_blank\">Audience targeting</a>.

URL

https://api.pinterest.com/v5/ad_accounts/{ad_account_id}/audiences

HTTP Method

POST

Response Type

Audience

Parameters

  • Path: ad_account_id String

    Unique identifier of an ad account.

  • Body: AudienceCreateRequest AudienceCreateRequest(AudienceCreateRequest)

    List of ads to create, size limit [1, 30]

audiences/list

Get list of audiences for the ad account.

URL

https://api.pinterest.com/v5/ad_accounts/{ad_account_id}/audiences

HTTP Method

GET

Response Type

audiences_list_200_response

Parameters

  • Path: ad_account_id String

    Unique identifier of an ad account.

  • Query: bookmark String

    Cursor used to fetch the next page of items

  • Query: order String

    The order in which to sort the items returned: “ASCENDING” or “DESCENDING” by ID. For received audiences, it is sorted by sharing event time. Note that higher-value IDs are associated with more-recently added items.

  • Query: page_size Integer

    Maximum number of items to include in a single page of the response. See documentation on <a href='/docs/reference/pagination/'>Pagination</a> for more information.

  • Query: ownership_type String

    Filter audiences by ownership type.

audiences/update

Update (edit or remove) an existing targeting audience.

URL

https://api.pinterest.com/v5/ad_accounts/{ad_account_id}/audiences/{audience_id}

HTTP Method

PATCH

Response Type

Audience

Parameters

  • Path: ad_account_id String

    Unique identifier of an ad account.

  • Path: audience_id String

    Unique identifier of an audience

  • Body: AudienceUpdateRequest AudienceUpdateRequest(AudienceUpdateRequest)

    The audience to be updated.
