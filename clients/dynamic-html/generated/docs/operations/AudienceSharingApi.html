This is the API

adAccountsAudiencesSharedAccounts/list

List all ad accounts and/or businesses that have access to a specific audience. The audience must be owned by the requesting ad account.

URL

https://api.pinterest.com/v5/ad_accounts/{ad_account_id}/audiences/shared/accounts

HTTP Method

GET

Response Type

ad_accounts_audiences_shared_accounts_list_200_response

Parameters

  • Path: ad_account_id String

    Unique identifier of an ad account.

  • Query: audience_id String

    Unique identifier of the audience to use to filter the results.

  • Query: account_type AudienceAccountType

    Filter accounts by account type.

  • Query: page_size Integer

    Maximum number of items to include in a single page of the response. See documentation on <a href='/docs/reference/pagination/'>Pagination</a> for more information.

  • Query: bookmark String

    Cursor used to fetch the next page of items

businessAccountAudiencesSharedAccounts/list

List all ad accounts and/or businesses that have access to a specific audience. The audience must either be owned by an ad account in the requesting business, or it must have been shared with the requesting business. If the requesting business is not the owner of the audience, only ad accounts owned by the requesting business will be returned.

URL

https://api.pinterest.com/v5/businesses/{business_id}/audiences/shared/accounts

HTTP Method

GET

Response Type

ad_accounts_audiences_shared_accounts_list_200_response

Parameters

  • Path: business_id String

    Unique identifier of the requesting business.

  • Query: audience_id String

    Unique identifier of the audience to use to filter the results.

  • Query: account_type AudienceAccountType

    Filter accounts by account type.

  • Query: page_size Integer

    Maximum number of items to include in a single page of the response. See documentation on <a href='/docs/reference/pagination/'>Pagination</a> for more information.

  • Query: bookmark String

    Cursor used to fetch the next page of items

sharedAudiencesForBusiness/list

Get a list of received audiences for the given business.

URL

https://api.pinterest.com/v5/businesses/{business_id}/audiences

HTTP Method

GET

Response Type

audiences_list_200_response

Parameters

  • Path: business_id String

    Unique identifier of the requesting business.

  • Query: bookmark String

    Cursor used to fetch the next page of items

  • Query: order String

    The order in which to sort the items returned: “ASCENDING” or “DESCENDING” by ID. Note that higher-value IDs are associated with more-recently added items.

  • Query: page_size Integer

    Maximum number of items to include in a single page of the response. See documentation on <a href='/docs/reference/pagination/'>Pagination</a> for more information.

updateAdAccountToAdAccountSharedAudience

From an ad account, share a specific audience with another ad account, or revoke access to a previously shared audience. Only the audience owner account can share the audience. The recipient ad account(s) must be in the same <a href='https://help.pinterest.com/en/business/article/create-and-manage-accounts'>Pinterest Business Hierarchy</a> as the business owner of the ad account.<br> This endpoint is not available to all apps.<a href='/docs/getting-started/beta-and-advanced-access/'>Learn more</a>.

URL

https://api.pinterest.com/v5/ad_accounts/{ad_account_id}/audiences/ad_accounts/shared

HTTP Method

PATCH

Response Type

SharedAudienceResponse

Parameters

  • Path: ad_account_id String

    Unique identifier of an ad account.

  • Body: SharedAudience SharedAudience(SharedAudience)

updateAdAccountToBusinessSharedAudience

From an ad account, share a specific audience with a business account, or revoke access to a previously shared audience. Only the audience owner account can share the audience. The recipient business account must be in the same business hierarchy as the business owner of the ad account.<br> This endpoint is not available to all apps.<a href='/docs/getting-started/beta-and-advanced-access/'>Learn more</a>.

Parameters

    Unique identifier of an ad account.

  • Body: BusinessSharedAudience BusinessSharedAudience(BusinessSharedAudience)

updateBusinessToAdAccountSharedAudience

From a business, share a specific audience with other ad account(s), or revoke access to a previously shared audience. <ul> <li>If the business is the owner of the audience, it can share with any ad account within the same business hierarchy.</li> <li>If the business is the recipient of the audience, it can share with any of its owned ad accounts.</li> </ul> This endpoint is not available to all apps.<a href='/docs/getting-started/beta-and-advanced-access/'>Learn more</a>.

URL

https://api.pinterest.com/v5/businesses/{business_id}/audiences/ad_accounts/shared

SharedAudienceResponse

Parameters

  • Path: business_id String

    Unique identifier of the requesting business.

  • Body: SharedAudience SharedAudience(SharedAudience)

updateBusinessToBusinessSharedAudience

From a business, share a specific audience with another business account, or revoke access to a previously shared audience. Only the audience owner can share the audience with other businesses, and the recipient business must be within the same business hierarchy.<br> This endpoint is not available to all apps.<a href='/docs/getting-started/beta-and-advanced-access/'>Learn more</a>.
